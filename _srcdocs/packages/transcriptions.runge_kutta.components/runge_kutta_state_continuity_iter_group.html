runge_kutta_state_continuity_iter_group.py¶

class dymos.transcriptions.runge_kutta.components.runge_kutta_state_continuity_iter_group.RungeKuttaStateContinuityIterGroup(**kwargs)[source]¶

Bases: openmdao.core.group.Group

This Group contains the k-iteration subgroup, the state advance component, continuity defect component. Given the initial value of each state at the beginning of the first segment, the times at the ODE evaluations, and the stepsize across each segment it will iterate to find the Runge-Kutta weights ‘k’ and the initial values of the states in all but the first segments to provide state continuity across the phase.

Notes

The objective can be scaled using scaler and adder, where

\[x_{scaled} = scaler(x + adder)\]

or through the use of ref/ref0, which map to scaler and adder through the equations:

\[ \begin{align}\begin{aligned}0 = scaler(ref_0 + adder)\\1 = scaler(ref + adder)\end{aligned}\end{align} \]

which results in:

\[ \begin{align}\begin{aligned}adder = -ref_0\\scaler = \frac{1}{ref + adder}\end{aligned}\end{align} \]
