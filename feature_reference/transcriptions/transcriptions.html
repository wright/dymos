Dymos implements several different transcriptions which are used to convert a continuous optimal control problem into an analogous nonlinear programming (NLP) problem. Currently, Dymos supports the following techniques:

  • High-Order Gauss-Lobatto Collocation
  • Radau Pseudospectral Method
  • Runge-Kutta Shooting Method
