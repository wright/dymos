Grid Refinement¶

Grid refinement is carried out via ph-refinement as proposed by Patterson, Hager, and Rao for Radau pseudospectral methods [PattersonHagerRao2015]. A similar method was shown for a Gauss-Lobatto transcription by Lei et al [LeiLiuLiYeShao2017].

Refinement is carried out iteratively in four steps:

  1. The error in each phase is computed for each phase in the solved problem.

  2. The errors are compared to the tolerance to determine which phases require refinement.

  3. Phases are refined via ph-refinement and the new required grid is computed.

  4. The problem is re-solved on the new grid and the process repeats until the error tolerance is met for all phases.
